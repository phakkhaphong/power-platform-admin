# เอกสารการสอน Power Platform Administration

**ระยะเวลา:** 1 วัน

## ภาพรวมหลักสูตร

หลักสูตรนี้แนะนำความสำคัญเชิงกลยุทธ์ของการบริหารจัดการ Power Platform โดยมุ่งเน้นจากการสนับสนุนแบบรับมือไปสู่การกำกับดูแลแบบเชิงรุก เราจะครอบคลุม "Three Pillars of Modern Administration" (Managed Governance, Security, and Operations) เพื่อวางกรอบการเรียนรู้ของวันนี้ เป้าหมายคือการสร้างสภาพแวดล้อมที่ปลอดภัยและเป็นนวัตกรรมใหม่ โดยหลีกเลี่ยงสถานการณ์ "Wild, Wild West" ของการพัฒนาที่ไม่สามารถควบคุมได้

**อ้างอิงหลัก:** [Microsoft Power Platform Admin Documentation](https://learn.microsoft.com/en-us/power-platform/admin/?wt.mc_id=ppac_inproduct_home)

## วิธีการใช้งานเอกสาร

แต่ละโมดูลแยกเป็น:
1. **README.md** - เนื้อหาทฤษฎี อธิบายแนวคิดหลักและแนวปฏิบัติที่ดี พร้อมสรุป Module (ใช้เป็น README ของแต่ละ Module)

## ข้อกำหนดเบื้องต้น

- เข้าใจพื้นฐานของ Microsoft 365 และ Cloud Services
- มีสิทธิ์เข้าถึง Power Platform Admin Center
- มีสิทธิ์ในการสร้างและจัดการ Environments (แนะนำ)
- ความเข้าใจพื้นฐานเกี่ยวกับ Microsoft Entra ID (เดิมชื่อ Azure AD)
